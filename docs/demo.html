ℹ️ About this Demo
CS2 CFG Configurator — Web Edition

This is the browser-only demo of the CS2 CFG Configurator project. It uses the File System Access API to let you pick your CS2 cfg folder and write files directly — no server, no upload.

What works in this web demo:

  • ✅ Bind Generator (visual, with presets)
  • ✅ Settings Editor (browse & tweak ConVars)
  • ✅ CFG Export → download or save directly to folder
  • ✅ Works 100% offline once loaded

What requires the Python app:

  • 🔒 Auto-detecting your Steam/CS2 installation path
  • 🔒 Reading existing .cfg files to import settings
  • 🔒 Profile management across sessions

Browser compatibility: Chrome 86+, Edge 86+. Firefox and Safari do not support File System Access API — use the download button instead.
