ℹ️ About – Web Edition
CS2 CFG Configurator — ported from Windows Python app to web with AI.

The original project was a Windows-only Python/CustomTkinter desktop app. I rewrote it as a fully browser-based web app using AI — every module is ported 1:1 so you get the exact same workflow without installing Python.

What works in the web version:

  • ✅ Dashboard with live stats
  • ✅ Buy Bind Builder (all weapons, grenades, armor)
  • ✅ Bind Switcher: Simple / Toggle / Hold / CFG-Exec
  • ✅ Settings Editor with search (38+ ConVars)
  • ✅ CFG Editor: read & write .cfg files directly via File System Access API
  • ✅ Command Viewer reference
  • ✅ Export: generate & save autoexec.cfg
  • ✅ LocalStorage persistence (survives page reload)

Only in the Python app:

  • 🔒 Auto-detection of Steam/CS2 installation path
  • 🔒 Multi-profile management saved on disk

File System Access API (direct folder write) requires Chrome 86+ or Edge 86+. Firefox/Safari: use the Download button.
