What it does

Everything you need for CS2 configs

⌨️

Bind Generator

Create and manage key bindings visually. No more manual cfg editing — just point, click and export.

🔧

Settings Editor

Browse all CS2 console variables with their default values, valid ranges and descriptions in one place.

📦

Modular Architecture

Built in Python with a clean module structure — easy to extend, contribute to, or fork for your own needs.

💾

CFG Export

Export your configuration directly to a valid .cfg file ready to drop into your CS2 folder.

🌐

Web Demo

Try the Bind Generator directly in your browser without installing anything.
