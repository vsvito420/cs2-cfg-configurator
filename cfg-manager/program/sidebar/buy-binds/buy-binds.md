
## 4. Buy Binds

muss überarbeitet werde macht kein sinn mit f1 tasten.. 

### Ziel
Automatischer Waffenkauf mit einer Taste (z.B. F1 = AK-47 + Helm + Molotov).

### Beispiel-Binds
```text
bind "f1" "buy ak47; buy vesthelm; buy molotov"
bind "f2" "buy m4a1; buy vesthelm; buy smokegrenade"
bind "f3" "buy awp; buy vesthelm"
bind "f4" "buy deagle; buy vest"
bind "f5" "buy hegrenade; buy flashbang; buy smokegrenade; buy molotov"
```

### Full Buy Script
```text
alias "fullbuy" "buy ak47; buy m4a1; buy deagle; buy vesthelm; buy vest; buy hegrenade; buy flashbang; buy smokegrenade; buy molotov; buy incgrenade; buy defuser"
bind "f6" "fullbuy"
```

### Programm-Feature
- Vordefinierte Buy-Presets (T, CT, Sniper, Nades)
- Custom Buy-Erstellung mit UI
- Automatisches Generieren der Bind-Syntax
